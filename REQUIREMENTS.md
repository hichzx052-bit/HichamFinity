# HichamFinity v2.0 — المتطلبات الكاملة

## المطور: Hicham

## تسجيل الدخول
- دخول بحساب تيك توك (Google/Facebook/يوزرنيم+باسوورد)
- يطلع صورة الحساب + اليوزرنيم + عدد المتابعين

## التشغيل
- تطبيق واحد — تلفون واحد
- يشتغل بالخلفية (ما يطفى) مثل TikFinity
- تفتح بث تيك توك + التطبيق شغال بالخلفية

## TTS (الأصوات)
- ينطق اسم اللي يدخل / يعلق / يهدي
- مرة وحدة بس (ما يتكرر!)
- تختار الصوت واللغة والسرعة

## فيديوهات تلقائية
- تختار فيديو لكل حدث:
  - X لايك → يطلع فيديو
  - X مشاهد → يطلع فيديو
  - هدية معينة → يطلع فيديو
  - زر يدوي → يطلع فيديو
- أنت تختار الفيديو لكل حدث

## تنبيهات الهدايا
- صوت + أنيميشن لكل هدية
- تخصيص لكل نوع هدية

## إحصائيات لايف
- عدد اللايكات / المشاهدين / التعليقات / الهدايا
- أكثر واحد هدى / علّق

## التصميم
- صور AI للأزرار والخلفيات
- دارك ثيم احترافي
- واجهة عربية RTL
- خط Tajawal

## ميزة خاصة
- 🏅 أول متابع / أول معلق / أول مهدي — تأثير خاص وإشعار مميز

## متطلبات تقنية
- Flutter + Node.js (Termux)
- الاتصال عبر WebSocket
- Background service (foreground notification)
- Android minSdk 24
